# Mémora — Mon français A2

یک وب‌اپ شخصی برای یادگیری واژگان فرانسوی سطح A2 با سیستم جعبهٔ لایتنر. پروژه کاملاً استاتیک است: حساب کاربری، بک‌اند و دیتابیس ندارد و تمام پیشرفت کاربر در `localStorage` همان مرورگر ذخیره می‌شود.

## امکانات

- جعبهٔ لایتنر ۵ مرحله‌ای با فاصله‌های ۱، ۳، ۷، ۱۵ و ۳۰ روز
- پاسخ تایپی از معنی فارسی به واژه یا عبارت فرانسوی
- پذیرش پاسخ بدون اکسنت و پخش تلفظ فرانسوی با صدای داخلی مرورگر
- داشبورد روزانه، هدف تمرین، دقت، استریک و نمودار فعالیت
- ۱۰۰ واژه و عبارت اصیل و کاربردی سطح A2 در ۱۰ موضوع
- افزودن، ویرایش و حذف واژه‌های شخصی
- Import سریع با فرمت `فرانسوی | فارسی | موضوع | مثال`
- جست‌وجو و فیلتر واژه‌ها بر اساس موضوع و جعبه
- خروجی و بازیابی پشتیبان JSON
- حالت روشن و تاریک، طراحی واکنش‌گرا و پشتیبانی آفلاین

## دربارهٔ واژگان Édito A2

بانک اولیه، مجموعه‌ای مستقل از واژه‌های رایج سطح A2 است و فهرست کامل کتاب دارای حق‌نشر را بازنشر نمی‌کند. برای افزودن واژه‌های نسخهٔ شخصی کتاب، از بخش «Importer» استفاده کنید:

```text
se souvenir | به یاد آوردن | Mes mots | Je me souviens de cette journée.
```

## اجرای محلی

به دلیل استفاده از ES Modules بهتر است فایل را از یک سرور محلی باز کنید:

```bash
npx serve .
```

سپس آدرسی را که در ترمینال نمایش داده می‌شود در مرورگر باز کنید.

## تست

```bash
npm test
```

## ذخیره‌سازی و حریم خصوصی

کلید اصلی برنامه در مرورگر `memora.francais.a2.v1` است. پاک‌کردن داده‌های سایت یا مرورگر، پیشرفت را حذف می‌کند؛ بنابراین از بخش Réglages به‌صورت دوره‌ای فایل پشتیبان JSON بگیرید. داده‌ها بین دستگاه‌ها همگام نمی‌شوند.

## استقرار

گردش‌کار `.github/workflows/pages.yml` پس از هر push روی شاخهٔ `main` تست‌ها را اجرا و نسخهٔ استاتیک را در GitHub Pages منتشر می‌کند.
